# Информационный киоск КГУ

Это веб-приложение для информационного киоска Костромского государственного университета (КГУ). Проект предоставляет удобный интерфейс для доступа к каталогу документов, новостей и другой полезной информации для студентов и посетителей университета.

## Технологии

- **Frontend**: React, TypeScript, Vite
- **Backend**: Deno (сервер на TypeScript)
- **Стилизация**: CSS
- **Данные**: JSON файлы для каталога и документов

## Структура проекта

- `src/` - исходный код frontend приложения
- `data/` - данные каталога и документов
- `public/` - статические файлы
- `server.ts` - серверный код на Deno
- `vite.config.ts` - конфигурация Vite

## Установка и запуск

### Prerequisites

- Node.js (для frontend)
- Deno (для сервера)

### Шаги установки

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите приложение в режиме разработки:
   ```bash
   npm run dev
   ```

3. Для запуска сервера (если необходимо):
   ```bash
   deno run --allow-net --allow-read server.ts
   ```

## Сборка для продакшена

```bash
npm run build
```

## Развертывание

Проект настроен для развертывания на Vercel или Render. Конфигурационные файлы: `vercel.json`, `render.yaml`.
